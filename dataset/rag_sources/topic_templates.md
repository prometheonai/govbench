# Topic → source templates (answer-free)

Use these templates to scale from the pilot to **all** questions. They are intended as **reusable “backbone sources”** per topic, plus a small set of “add if needed” sources for edge-cases.

Hard rule: **never use** sources with `/vraag-en-antwoord/` in the URL.
